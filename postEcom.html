Laragigs



The web app is a fashion and electronics store.

User can add items to the cart and complete payments with paypal either as a logged in authenticated user or as a guest user.

The total number of items in the cart and the total price updates automatically as you add more items to the cart, the log in and register routes allows users to for easier session managment

Technologies:
  • - Python
  • - Django
  • - Vanilla JavaScript
  • - SQLite
